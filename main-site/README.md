# SG MRT Alerts

Real-time Singapore MRT situational awareness at your fingertips.

## Features

- 🚨 **Train Alerts** — Live disruptions and delays across all MRT/LRT lines
- 🛗 **Lift Maintenance** — Accessibility-focused lift outage tracking, filterable by station
- 👥 **Crowd Now** — Real-time crowdedness per station (updated every 10 min)
- 📅 **Crowd Forecast** — 30-minute interval crowd forecasts for trip planning
- 🎨 **7 themes** — Classic green, Rosy, Lavender, Butter, Lilac, Sky, Frost
- 📱 **PWA** — Installable, offline-capable

## Notes

- Crowd tabs require selecting a line first (mandatory `TrainLine` param)
- Alerts and Lifts tabs auto-refresh every 60 seconds
- Crowd Now auto-refreshes every 60 seconds when a line is selected
- Alert status 1 = Normal/Minor Delays; status 2 = Disrupted/Major Delays
